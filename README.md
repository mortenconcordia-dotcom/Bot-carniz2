# Telegram Cornice Bot — bothost.ru (Docker)

## Что это
Telegram-бот-калькулятор карнизов (polling, без вебхука).

## Как деплоить на bothost.ru
### Вариант 1: через Git репозиторий
1) Создайте репозиторий на GitHub и загрузите файлы этого проекта (важно: **Dockerfile** в корне).
2) В панели bothost укажите ссылку на репозиторий.
3) В переменных окружения задайте:
   - BOT_TOKEN=ваш_токен_от_BotFather
4) Нажмите "Пересобрать/Deploy".

### Вариант 2: загрузка файлов (если доступно)
Загрузите файлы проекта так, чтобы Dockerfile лежал в корне, затем пересоберите контейнер.

## Локальный запуск (VS Code)
pip install -r requirements.txt
export BOT_TOKEN="..."
python bot.py

## Команды
- /start — меню
- /test — контрольные значения

## Логика (как калькулятор)
- К центру: L = X - 15.2
- Слева-Направо: L = X - 11.6
- Схема: N = ceil(L/300), S = round(L/N, 1) -> S выводится N раз
- Бегунки: even_up(ceil(X/8))
- Крючки: бегунки + 10
- Крепления: ceil(X/100) + 1
